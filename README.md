# フラットタイプカーポート高さ計算機

柱間寸法と勾配から  
柱間高低差を計算する  
現場向けWEBツールです。

## 使い方

ブラウザで公開URL、または `index.html` を開くだけで利用できます。アプリ化やビルドは不要です。

## 計算式

```text
柱間高低差(mm) = tan(勾配°) × 柱間寸法(mm)
```

表示値は現場で確認しやすいように整数mmへ丸めています。

## 拡張方針

`js/app.js` の `tools` オブジェクトに計算機単位で定義を追加できる構成にしています。将来的にフェンス実高さ計算機、階段高さ計算機などを追加する場合も、既存ロジックと画面を分けて拡張できます。
